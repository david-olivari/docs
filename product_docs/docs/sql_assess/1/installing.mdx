title: Installing EDB SQL Assess
navTitle: Installing
---

The EDB SQL Assess utility can be installed using the following procedure:

1. Download the ZIP file from <this location>.

1. Unzip the file in a folder to extract a JAR and a SQL script as listed below:

   ```text
   assessSQL.jar
   edb_queries_extractor.sql
   ```

Installation is complete.
